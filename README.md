# Monte Carlo Tree Search and Minimax Search for Connect 4.

Empirical comparison of Monte Carlo Tree Search (MCTS) and Minimax Search for the solved, perfect-information Connect X. For more information, see [distill post](https://mbaranr.github.io/blog/2024/c4_search/).

## Run

```
$ cd c4_search
$ pip install -r requirements.txt
$ python3 main.py
```
